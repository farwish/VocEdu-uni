# vocedu-uni

## 概览

项目是使用uni-app官方编辑器 HBuilder 建立的默认项目。

在此基础上：
增加了 http/ 目录放置有 api 列表和公共请求方法；
增加了 store/ 目录放置有 vuex 系列文件；
以上都在 main.js 文件中注册在 Vue 之上。

其余是 uni-app 项目结构。
